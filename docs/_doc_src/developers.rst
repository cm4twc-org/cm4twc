Developer Guide
===============

This guide is for people interested in extending the technical
capabilities of the framework.

.. warning::

   Not available yet.

The transfers that make up the framework (shown in Table x and Fig y) are defined in the unifhy/component.py file. To add new transfers to the framework it is simply a case of adding the name and details of the transfer to the outwards section of the component the transfer is coming from and the inwards section of the component the transfer is going to.
