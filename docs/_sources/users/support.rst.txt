Support
=======

Support for users and contributors is available through
`GitHub Discussions <https://github.com/unifhy-org/unifhy/discussions>`_.
Use this support page to find answers to previously asked questions,
to ask questions of your own, and to suggest ideas to improve `unifhy`.

To report bugs, please use `GitHub Issues <https://github.com/unifhy-org/unifhy/issues>`_.
